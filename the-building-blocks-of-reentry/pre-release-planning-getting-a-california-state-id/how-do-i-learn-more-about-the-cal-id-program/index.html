How do I learn more about the CAL-ID Program?

If you have questions about the CAL-ID program, call Nikita Singh, CAL-ID Coordinator, In-Prison Unit, Division of Rehabilitative Programs, California Department of Corrections and Rehabilitation (CDCR), at the following phone number: 1-916-327-3352, and/or talk to your correctional counselor.

If you have Internet access, you can also view CDCR’s Fact Sheet on the CAL-ID program online: http://www.cdcr.ca.gov/rehabilitation/docs/Factsheets/OS-IP-Factsheet-CalID-July2015.pdf
